Dizerodha LLP Website
====================

This is the official website for Dizerodha LLP, providing seamless online services across India.

Project Purpose:
- Offer central and state-specific services (e.g., PAN, Passport, ITR, FSSAI, etc.)
- Enable users to contact and enquire online from the comfort of their home
- Built with HTML, CSS, JS, and EmailJS for contact forms (no backend required)

How to Use:
- Open index.html in your browser to view the website
- The contact form uses EmailJS (see index.html for configuration)
- For customization, edit the HTML files and assets as needed
